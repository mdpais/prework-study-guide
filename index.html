✨ Open the Console to See What's Happening ✨

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout main switches to the main branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A stages all the files
  • git commit -m "message" commits the files and includes the message
  • git pull origin main makes sure local branch is upto date before pushing changes
  • git push origin branch-name pushes changes to remote repository

Other

  • ls displays all files in the folder
  • touch filename.fileextension creates file with the name and extension
  • code . opens the Visual Studio Console

JavaScript

  • Control flow is the order in which a computer executes code in a script.
  • var x=0 declares variable x
  • '' contains text strings
  • var y=[] contains an array within []
  • , separates array elements
  • for loop (var x = 0; x < y.length; x++)
    • .length is the length of the array
    • x++ increases the variable by 1 each time until .length is met
